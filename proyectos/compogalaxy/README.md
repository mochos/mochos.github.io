### Requisitos para jugar

* Haber sido invitado por Compo
* Tener usuario y clave asignado

### Paso a paso para entrar al servidor

1. Instala [Starbound](https://store.steampowered.com/app/211820/Starbound/)
2.  Instala los [mods requeridos](https://steamcommunity.com/sharedfiles/filedetails/?id=3537094884) para el servidor.<br>

    <figure><img src=".gitbook/assets/00-Suscribirse a todos.png" alt=""><figcaption></figcaption></figure>
3. Instala y configura la [extensión para chat de voz](ayuda.md#extension-para-chat-de-voz).
4. [Entra al servidor](ayuda.md#entrar-al-servidor)
