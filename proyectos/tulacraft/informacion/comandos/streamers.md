# 🎥 Streamers

### Anuncios

Los streamers pueden anunciar en el chat cuando estén en directo por medio de un comando sencillo. Primero debes solicitar a la administración que te habiliten este permiso.

`/twitch set <usuario de twitch>` Registras el nombre de usuario de tu canal de Twitch.

`/tiktok set <usuario de twitch>` Registras el nombre de usuario de tu canal de TikTok.

`/twitch` `/tiktok` Haces el anuncio en todo el servidor de que estás en directo.

{% hint style="info" %}
Si eres streamer y no tienes permiso para usa usar estos comandos, ponte en [contacto con el staff](./#ayuda-del-staff).
{% endhint %}
